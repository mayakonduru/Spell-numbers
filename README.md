# Estonian Number Speller

This program converts numbers into their Estonian word representation. It supports numbers from 0 up to 999,999 and writes the results to an output file. 

# Features
- Converts numbers to Estonian words.
- Handles numbers from 0 to 999999
- Supports input from a text file (one number per line)
- Outputs the spelled numbers to a specified file in a readable format.

# Installation 
- Clone the repository or download the script spell-numbers.py
